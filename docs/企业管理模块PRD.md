# 企业管理模块 PRD

## 1. 模块概述

企业管理模块用于后台管理员创建、编辑和查看企业客户信息。每个企业拥有独立的登录凭证、溢价系数和月账期额度。

### 核心能力
- 企业账号的 CRUD 管理
- 自动生成唯一企业 ID
- 地区号与国家/币种联动
- 完整的字段校验体系
- 密码脱敏显示

---

## 2. 数据模型

| 字段 | 类型 | 说明 | 规则 |
|------|------|------|------|
| id | string | 企业ID | 格式：`ENT-YYYYMMDD-XXX`，系统自动生成，不可修改 |
| name | string | 企业名称 | 2-50字符，禁止 `< > ' " & \`，全局唯一 |
| countryCode | string | 地区号 | 下拉选择：+852/+66/+86/+65/+60 |
| phone | string | 登录手机号 | 纯数字，位数由地区号决定，同地区唯一 |
| password | string | 登录密码 | 8-20位，须同时包含字母和数字 |
| country | string | 国家 | 由地区号自动确定 |
| currency | string | 币种 | 由地区号自动确定 |
| premiumRate | number | 溢价系数 | 范围 1.00-3.00，精度 2 位小数 |
| creditLimit | number | 月账期额度 | 整数，0-10,000,000 |
| usedCredit | number | 已用额度 | 系统计算，不可手动修改 |
| createdAt | string | 创建时间 | 系统自动生成 |

## 4. 页面交互说明

### 4.1 企业列表页 `/admin/enterprises`

**列定义（从左到右）**：
1. 企业ID — `font-mono` 等宽字体
2. 企业名称
3. 登录手机号 — 显示格式：`{地区号} {号码}`
4. 登录密码 — 默认显示 `••••••••`，点击眼睛图标切换明文/脱敏
5. 国家
6. 溢价系数
7. 月账期额度 — 显示格式：`{币种} {金额}`
8. 已用额度
9. 创建时间
10. 操作 — 更改信息 / 交易明细 / 订单记录

**搜索**：支持按企业ID、名称、手机号搜索。

### 4.2 创建企业页 `/admin/enterprises/create`

**交互流程**：
1. 填写表单（所有字段均有校验）
2. 选择地区号 → 自动锁定币种、调整手机号校验
3. 点击「创建账号」→ 校验通过后弹出 Modal 展示生成的企业ID
4. 点击「返回列表」回到列表页

**表单字段**：企业名称、登录手机号（地区号+号码）、登录密码（`Input.Password`）、溢价系数、月账期额度（币种+金额）

### 4.3 编辑企业页 `/admin/enterprises/[id]/edit`

与创建页对称，差异点：
- 表单顶部展示企业ID（`disabled Input`，不可修改）
- 唯一性校验（企业名称、手机号）排除当前企业自身
- 预填充现有数据

---

## 5. 校验规则详细说明

### 企业名称
- 必填
- 长度：2-50 个字符
- 禁止字符：`< > ' " & \`
- 全局唯一（编辑时排除自身）

### 登录手机号
- 必填
- 只允许数字
- 位数根据地区号动态校验（见映射表）
- 同地区号下唯一（编辑时排除自身）
- 切换地区号时自动重新校验

### 登录密码
- 必填
- 长度：8-20 个字符
- 必须同时包含字母和数字
- 创建/编辑页使用 `Input.Password` 组件（支持显示/隐藏）
- 列表页默认脱敏显示

### 溢价系数
- 必填
- 类型：数字
- 范围：1.00 ~ 3.00
- 精度：2 位小数

### 月账期额度
- 必填
- 类型：整数
- 范围：0 ~ 10,000,000
- 币种由地区号自动确定，不可手动选择

---

## 6. 企业 ID 生成规则

格式：`ENT-YYYYMMDD-XXX`

- `ENT`：固定前缀
- `YYYYMMDD`：创建日期
- `XXX`：三位序号，同日内自增（001, 002, ...）

示例：`ENT-20260330-001`
